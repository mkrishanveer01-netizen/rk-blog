❓ Kaise Use Karein

1. Chat mein type karo — jaise "500 kharcha chai" ya "1000 profit aaj". App khud samajh jayega ye kharcha hai ya profit.

2. Agar sirf number likho (jaise "500") to app poochega Kharcha ya Profit — button dabao, ho gaya.

3. Quick buttons se bhi bina type kiye seedha amount daal sakte ho.

4. Chart par kisi bhi din ke bar par click karo — us din ki poori detail dikhegi.

5. Reminder on karo taaki agar aaj entry na daali ho to yaad dilaya jaaye (sirf tab jab yeh page khula ho).

6. Report Download se apna data CSV file mein save kar sakte ho — Excel mein khul jaati hai.

Koi bhi galat entry ho jaaye to Recent Entries list mein 🗑️ button se delete kar do.
