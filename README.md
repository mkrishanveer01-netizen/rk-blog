## Kharcha-Profit Manager

Rozana kharcha aur profit chat mein type karke manage karne ke liye ek simple, beautiful dashboard: [`kharcha-dashboard/index.html`](./kharcha-dashboard/index.html)

**Kaise use karein:**
- File ko seedha browser mein khol lo (double-click), ya GitHub Pages se link share karo.
- Chat box mein type karo jaise `"500 kharcha chai ke liye"` ya `"1000 profit aaj"` — app khud samajh kar add kar dega.
- Sirf number (jaise `"200"`) likhoge to app poochega Kharcha ya Profit — button dabao, ho gaya.
- Dashboard mein aaj/mahine ka total, 14-din ka chart (kisi bhi din par click karke detail dekho), recent entries, aur CSV report download sab available hai.
- Data sirf usi browser mein save hota hai (koi server/login nahi chahiye) — isliye backup ke liye CSV download karte raho.
- Koi bhi 10th class student bina kisi technical knowledge ke use kar sakta hai.
